# SkyCast

A real-time weather application with animated, condition-based backgrounds (rain, snow, clouds, sun).

## Features
- Real-time weather data using OpenWeatherMap API
- 5-day forecast
- Geolocation support ("Use my location")
- Recent search history
- Dynamic background based on weather condition
- Animated weather effects (rain, snow, clouds, sun glow)
- Smooth transitions with Framer Motion

## Tech Stack
- React (Vite)
- Tailwind CSS
- Framer Motion
- Axios
- OpenWeatherMap API

## Getting Started
```bash
npm install
npm run dev
